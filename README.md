# perfect_guess_game

**Introduction**

The Perfect Guess Game is an interactive text-based game where the player has to guess a randomly chosen number within a specified range. The objective is to guess the correct number with the fewest attempts possible. It's a fun and challenging game that tests your guessing skills.

**Features**

Random Number Generation: The game generates a random number within a specified range for the player to guess.
Player Interaction: Players input their guesses and receive feedback on each attempt.
Attempts Count: The game keeps track of the number of attempts taken to guess the correct number.
Win Condition: The player wins when they guess the correct number.

**Usage**

Run the program.
The game generates a random number within a predefined range.
The player is prompted to input their guess.
The program provides feedback on whether the guess is too high or too low.
Continue guessing until the correct number is guessed.
The game displays the number of attempts it took to guess correctly.
